# Cyphra

Un generador de contraseñas seguro y personalizable escrito en Python.

## Características

- Generación de contraseñas con longitud personalizable (8 a 32 caracteres).
- Incluye letras, números, símbolos y espacios.
- Opción de visualizar u ocultar la contraseña generada.

## Uso

Ejecuta el script con Python:

```bash
python generator.py
```

## Requisitos

- Python 3.x
- `cryptography` (opcional para futuras implementaciones de cifrado)
